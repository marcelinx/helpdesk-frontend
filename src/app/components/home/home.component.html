HelpDesk Uma aplicação de Helpdesk simples!

Bem-vindo ao sistema de Help Desk desenvolvido por João Marcelino. Aqui, apresentamos um sistema de gerenciamento de chamados que simula um Help Desk simples, construído com Angular 12 e Spring Boot 2. Este sistema está hospedado na nuvem. O objetivo central deste projeto é abordar conceitos essenciais, tais como Single Page Applications (SPAs), autenticação e autorização com tokens JWT, entre outros. Para explorar o código-fonte do projeto, você pode acessar o link para o repositório no GitHub disponível no menu lateral.

Sinta-se à vontade para testar o sistema e compartilhar suas sugestões conosco.
